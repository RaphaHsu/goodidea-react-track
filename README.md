# 天瓏書籍搜尋器
工作室書單查詢練習，使用 [好想天瓏書單API](https://www.ptt.cc/bbs/Soft_Job/M.1539751254.A.E2C.html)

Demo Site: https://oscar60310.github.io/goodidea-react-track/Q4/

# Requirement
- node 10
- yarn

# Getting Started
- Install packages
  
  Run `yarn`
- Start development server

  Run `yarn start`
- Open http://localhost:8080
